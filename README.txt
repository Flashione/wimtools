WimTools Customer Restore

1. Run Create-WimTools-USB.cmd as Administrator.
2. Select the USB stick disk number.
3. Type CREATEUSB to erase and prepare the USB stick.
4. Copy the content of the WindowsPE folder to the BOOT partition.
5. Copy the WimTools folder to the WIMTOOLS partition.
6. Boot the target device from the USB stick.
7. Select the target disk.
8. Type ERASE to start the restore.

WARNING:
The selected target disk will be completely erased.
